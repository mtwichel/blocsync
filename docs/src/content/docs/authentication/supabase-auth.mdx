## Install

Import
[supabase_blocsync_authentication_provider](https://pub.dev/packages/supabase_blocsync_authentication_provider)

```base
flutter pub add supabase_blocsync_authentication_provider
```

## Configure

Configure Blocsync with Supabase Auth

```dart
import 'package:blocsync/blocsync.dart';
import 'package:supabase_flutter/supabase_flutter.dart';
import 'package:supabase_blocsync_authentication_provider/supabase_blocsync_authentication_provider.dart';

void main() async {
  WidgetsFlutterBinding.ensureInitialized();

  await Supabase.initialize(
    url: 'https://your-project.supabase.co',
    anonKey: 'your-anon-key',
  );

  BlocSyncConfig.authProvider = SupabaseAuthProvider();
  BlocSyncConfig.apiClient = ApiClient(
    baseUrl: Uri.parse('https://your-server.com'),
    apiKey: 'your_api_key',
  );

  runApp(MyApp());
}
```
